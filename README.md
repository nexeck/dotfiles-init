# dotfiles-init

Automated initial bootstrap/setup for macOS systems.

## Usage

### One-liner (Remote Execution)

To download and execute the initialization script directly from GitHub without cloning the repository, run:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/nexeck/dotfiles-init/main/init.sh)"
```

### Local Execution (Cloned Repository)

If you have cloned this repository locally, you can run the script with:

```bash
./init.sh
```
